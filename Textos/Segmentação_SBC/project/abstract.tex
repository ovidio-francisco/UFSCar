\begin{abstract}
% 
% 
The text segmentation task consists of split a text in parts with relatively independent meaning, so that each segment contains a subject.
% 
% 
Many researches evaluates they text segmentation methods using long texts as concatenation of documents from several subjects or transcriptions of discourse and multipart meetings.
% 
This article concentres in the main algorithms of the literature employing them in the context of the meeting minutes in Portuguese, which besides the language less studied with this method, has a writing style more formal and succinct.
%
%
Finally, we presents a method to segment the meeting minutes with similar performance to others works.


\end{abstract}
